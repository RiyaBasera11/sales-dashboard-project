# Sales Dashboard Project

Built a Sales Dashboard using Power BI to track monthly revenue, top-selling products, and regional performance.
Created interactive visuals that help identify sales trends and growth opportunities.

## Tools Used
- Power BI
- Excel

## Dashboard Preview
![Dashboard Preview](./Screenshot%202026-05-02%20234936.png)

## Key Insights
- Total Sales: 2.30M across all regions
- Technology category generates highest profit  
- Phones and Chairs are top-selling sub-categories
- West region contributes 31.58% of total sales
